VERSA 1.0 · Vendor-neutral Extensible Repo Spec for Agents

VERSA translates lessons from AGENTS.md, CLAUDE.md, GEMINI.md, and Cursor rules into a single portable schema. Author once, ship everywhere, stay future-proof.

  • Interoperable
    Works with IDEs, CLIs, hosted copilots.
  • Extensible
    Profiles, extensions, forward-compatible metadata.
  • Secure
    Deny/ask/allow permissions, secret bindings, redaction policies.
  • Accessible
    CodeVibe system enforces AAA contrast across UI.
Read SPEC.md Install CLI

Merge order

  1. 1. `.ai/settings.json`
  2. 2. `.ai/profiles/<name>.settings.json` (when `AI_PROFILE` is set)
  3. 3. `~/.ai/settings.json` (keys flagged `user_overridable`)
  4. 4. Runtime flags & ephemeral overrides

Permissions

deny → ask → allow

The spec mandates secret bindings via environment or keychain only, with automatic redaction guidelines for knowledge ingestion.
